# Concise Analytical Report Construction

This skill provides a protocol for generating compressed analytical syntheses. It preserves core reasoning and structural integrity while minimizing redundancy and exhaustive expansion.

## Core Procedural Workflow

### Phase 0: Pre-requisite Validation
1.  **Verify Analysis Baseline**: Confirm that a `perceptual-decomposition` analysis has been completed and that primary signals and patterns are identified in the context.

### Phase 1: High-Value Context Distillation
1.  **Extract Primary Signals**: Identify the most significant structures, patterns, and anomalies from the perceptual analysis data.
2.  **Filter Noise**: Remove granular decomposition and low-impact observations.
3.  **Group Observations**: Aggregate related surface characteristics into high-level themes.

### Phase 2: Compressed Synthesis
1.  **Draft Executive Summary**: Formulate a system identity hypothesis and core behavioral assessment (10-20% of report).
2.  **Construct Analysis Sections**: Build the key observations and structural/behavioral analysis sections using high-abstraction language.
3.  **Perform Assessment**: Identify the strongest evidence-supported interpretations of patterns and intent.
4.  **Enforce Size Constraints**: Aim for 40-200 lines. Prioritize depth of reasoning over length.

### Phase 3: Structural Presentation
1.  **Apply Markdown Hierarchy**: Use organized headers, bullet lists, and tables for rapid scanning.
2.  **Utilize Blockquotes**: Highlight major conclusions and system identity hypotheses.
3.  **Maintain Traceability**: Ensure that even under compression, the reasoning path from observation to conclusion is clear.

### Phase 4: Standardized Export
1.  **File Generation**: Use `scripts/export_concise_report.py` to save the report as a `.md` file following the `concise_analysis_[subject]_[timestamp].md` convention.

## Reasoning Constraints
- **Intelligent Compression**: Prioritize structural significance and conceptual importance.
- **Analytical Coherence**: Never sacrifice reasoning integrity for the sake of brevity.
- **Probabilistic Ranking**: Present findings ranked by confidence level.
